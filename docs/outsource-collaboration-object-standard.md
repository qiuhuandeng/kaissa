# 外采产品协同对象与页面归属标准

## 适用范围

本标准适用于外采成品产品、外采团期、外采订单确认和供应商对账结算原型。目标是明确供应商协同端与凯撒内部后台的对象边界，避免把供应商维护动作、凯撒包装动作、订单确认动作和财务动作混在同一个页面。

## 角色边界

| 角色 | 主要入口 | 负责对象 | 不负责 |
| --- | --- | --- | --- |
| 供应商 | 供应商协同端 | 供应商原始产品、供应商团期、订单接待确认、账单确认 | 凯撒销售包装、凯撒销售价、渠道开售、付款执行、NC |
| 外采产品运营 | 产品中心 > 外采产品 > 代理产品 | 查看供应商提交内容、包装凯撒代理产品、控制外采销售团期开售/停售 | 维护供应商自己的产品档案、进入自营履约中心 |
| 销售/客服 | 销售中心 | 查看订单供应商确认状态、处理客户改期/取消/退款入口 | 修改供应商原始产品或供应商团期 |
| 财务 | 财务模块 | 应付、付款执行、对账结算、发票、NC | 供应商接单确认、凯撒包装 |

## 对象拆分

| 对象 | 维护方 | 主要字段 | 页面归属 |
| --- | --- | --- | --- |
| 供应商原始产品 | 供应商 | 产品名称、类型、行程、供应商报价、费用规则、退改规则、素材、可供采状态 | 供应商协同端 > 我的产品 |
| 凯撒代理产品 | 外采产品运营 | 包装名称、销售卖点、展示素材、销售价规则、渠道说明、销售状态 | 产品中心 > 外采产品 > 代理产品 / 详情 / 方案包装 |
| 供应商团期 | 供应商 | 出发日期、航次、班期、出行日期、可售状态、供应商底价、报名截止、价格变更 | 供应商协同端 > 我的团期 |
| 凯撒外采销售团期 | 外采产品运营 | 是否开售、销售价、渠道可售、停售、订单占用 | 产品中心 > 外采产品 > 代理产品 > 可代理团期 |
| 外采订单确认 | 供应商回传，凯撒查看/补录 | 供应商确认状态、确认号、确认备注、保留时效、补差金额、无位建议 | 供应商协同端 > 订单确认；销售中心 > 订单详情 |
| 供应商结算单 | 凯撒生成，供应商确认 | 结算周期、订单明细、应结金额、差异说明、发票信息 | 供应商协同端 > 对账结算；凯撒财务模块 |

## 页面归属规则

1. 代理产品列表是凯撒内部入口，只处理供应商提交后的外采来源、包装、团期销售控制和停售。
2. 外采方案包装左侧展示供应商原始内容，只读；右侧展示凯撒包装内容，可编辑。
3. 外采团期只从代理产品列表的操作列进入，展示当前产品对应的凯撒外采销售团期，不进入自营团期列表。
4. 供应商产品上架、供应商团期维护、供应商订单接单确认均在供应商协同端完成。
5. 订单详情只承接外采供应商确认结果和补录入口，不维护供应商原始产品和供应商团期。
6. 对账结算中供应商只确认账单、提出差异和提交发票信息；应付、付款执行、回单和 NC 仍由凯撒财务模块处理。

## 状态口径

### 供应商原始产品状态

| 状态 | 含义 |
| --- | --- |
| 草稿 | 供应商保存但未提交凯撒 |
| 已提交 | 已提交凯撒等待处理 |
| 凯撒退回 | 凯撒要求补充或修改 |
| 已采用 | 已被凯撒包装或引用 |
| 停止供采 | 供应商不再供采 |

### 凯撒外采来源状态

| 状态 | 含义 |
| --- | --- |
| 供应商已提交 | 供应商提交了新的原始产品或报价 |
| 供应商有更新 | 供应商更新了产品、团期或价格，凯撒需复核 |
| 凯撒已包装 | 已形成凯撒销售包装 |
| 待补资料 | 凯撒已退回供应商补充资料 |

### 供应商团期状态

| 状态 | 含义 |
| --- | --- |
| 可售 | 供应商确认可继续接单 |
| 少量 | 余位、余舱或房量紧张 |
| 售罄 | 供应商已无可售量 |
| 停售 | 供应商停止供采 |
| 价格变更待确认 | 供应商价格发生变化，待凯撒复核 |

### 订单供应商确认状态

| 状态 | 含义 |
| --- | --- |
| 待确认 | 已发送供应商，等待回复 |
| 已确认 | 供应商确认接待并回传确认号或确认记录 |
| 无位 | 供应商无法接待，可给候补或改期建议 |
| 需补差 | 供应商要求补差价或改服务标准 |
| 已取消 | 订单对应外采接待已取消 |
